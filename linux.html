I completed the LPI Linux Essentials course, found on Linux Academy. This course takes you through the basics of Linux, including a background in the history of Linux as well as several of the different commonly used distributions. It also covers security concepts and the components of a Linux system.

I already had a good grounding in Ubuntu, but it was helpful for me to be exposed to some of the differences between Ubuntu and Debian.

Course Notes

  • This course could have been better, in that it was difficult to follow along with the videos. The videos are mostly standalone, without content that you can do on your own setup.
  • Watch the videos at as fast a speed as you can process and you’ll get more done faster. Especially the first few videos that don’t convey much substantive information, but more information about the class, I was able to watch at 1.75x and get through pretty quickly.
  • Installing CentOS on VirtualBox
    • I was unable to get my mouse working within CentOS during the installation process. However, I was able to complete the installation by using keyboard commands (tab, spacebar, enter, esc). Oddly enough, my scroll wheel worked, but no mouse pointer showed up anywhere.
  • Fixing the Mouse
    • Mouse initially appeared to not function on the guest OS, even when I tried to attach it by going to Devices->USB Wireless Mouse
    • As it turned out, the mouse was invisible. This is likely do to my using a SURFACE PRO. The fix:
      • Shutdown the guest OS
      • Download and install the VirtualBox Extension Pack: (https://www.virtualbox.org/wiki/Downloads)
      • With the guest OS shut off, go to Settings->System>Motherboard Tab, then click the dropdown for Pointing Device and set it to “USB Multi-Touch Tablet”.
      • Power on the Guest OS. On the Guest OS window, click Input->Enable Mouse Pointer Integration
      • Mouse should now be visible.
