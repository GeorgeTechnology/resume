What is a VPN?

A virtual private network (VPN) is one of the best tools an average user can deploy to protect their information while in public. This technology was originally developed as a way for remote users to access resources on local networks - such as a traveling employee needing to access an internal file share on a corporate network. This is accomplished by creating an encrypted tunnel between the two points. Because of this encryption, the traffic between the user’s device and the remote network is entirely indecipherable to the networks this traffic traverses.

Simply put, if a user is in public, enabling a VPN hides her traffic from all devices on the same public network and protects her information from being stolen or mined.

Why is this useful to me?

As something of a privacy nerd myself, I had already been paying for a VPN service. However, even paid VPN services have been known to mine and sell data of their users. By building my own VPN server, I ensure the maximum amount of protection for my traffic, outside of building my own data center.

Install Notes

  • You will need to configure a DigitalOcean server, which wasn’t explained in instructions.
  • You will need to add a sudo user. This is a basic linux task, but it isn’t explained in instructions
  • Because I didn’t have a GUI interface on the remote server, I had to download the algo folder to my virtual machine, and then use SCP to copy contents of the folder to remote digitalocean server.
  • When you execute the the sudo commands and python commands, make sure to do them from the algo folder or they won’t work
  • It is possible to install the OpenVPN server on the same DigitalOcean instance as the Algo server. Just install Algo first, and then start the instructions for OpenVPN.
  • SNAPSHOTS ARE YOUR FRIEND. Take them often. You often won’t know that you’ve royally messed something up until an hour after you did it, and being able to revert to a 2-hour old snapshot is much better than having to start at square one again.
  • If you don’t want to pay for the VPN server, it is possible to completely delete your digital ocean account after the free trial period has expired. Then, use the free trial link found in the Algo instructions to re-create your account.
    • You’ll have to set up a whole new server from scratch, but once you’ve done it a couple times this can be done in 20 minutes.
